# 个人 GitHub Pages 网站

这是一个可直接发布到 GitHub Pages 的静态个人主页。

## 修改内容

- 在 `index.html` 中替换你的名字、简介、项目、文章和联系方式。
- 在 `styles.css` 中调整颜色、间距和字体。

## 发布到 GitHub Pages

1. 在 GitHub 创建一个仓库，推荐命名为 `你的用户名.github.io`。
2. 把本文件夹里的 `index.html`、`styles.css`、`README.md` 推送到该仓库。
3. 打开仓库的 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main` 和 `/root`。
5. 保存后等待一会儿，网站会出现在 `https://你的用户名.github.io/`。

如果你把网站放在普通仓库里，也可以用 GitHub Pages 发布，网址通常会是
`https://你的用户名.github.io/仓库名/`。
